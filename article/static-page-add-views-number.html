静态网站添加统计访问点击量

该文章创建于2019-12-26，最后修改于2019-12-26

一个页面如果要实现可以统计访问点击量，那么一定是需要后台数据库记录数据的

但是大多数人的博客系统都是托管于github page或者是coding page这一类代码托管网站的，这就导致我们的博客页面都是静态页面，很显然，静态页面是不具有数据库的

那么，我们应该是用什么来统计访问量呢？

百度统计？谷歌统计？可能这是你立马能想到的几种解决办法，但是，这些网站统计，必须是网站被收录才能使用的，所以只能无奈地PASS掉了

现在我给大家介绍下我使用的统计方式，emmm，顺便说一下，这个方式是我扒gitbook统计阅读量插件的源码找到的...

方法就是这个：hit-counter

这是一个开源的访问统计项目，使用方式也很简单，在你的页面公共js里加下面的代码就可以了

let targetUrl = window.location.href;
$.ajax('https://hitcounter.pythonanywhere.com/count', {
    data: {url: targetUrl},
}).then(count => $("#num").html(count));

它会获取你当前页面的url，异步发送给这个网址，然后返回给你统计量，你只需要在你的页面上设置一个含有id为num的元素就可以接收这个访问统计量了

是不是很简单呢~
