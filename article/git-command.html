Git常用命令

该文章创建于2020-6-25，最后修改于2020-2-23

命令行 含义
git config [--global] user.name "xxx"
git config [--global] user.email xxx@xxx.com		 创建【公共的】用户名和邮箱
git init 初始化本地仓库
git add . 将文件全部添加到暂存区
git commit -m "xxx" 提交文件，并添加xxx描述信息
git clone [URL] [xxx] 从远程仓库URL复制文件到xxx目录下
git remote add [name] [URL] 给远程仓库URL起一个名字并以键值对保存
git remote show [name] 查看该名字对应的远程仓库URL
git remote rm [name] 删除改名字对应的远程仓库URL
git remote rename [name] [new-name] 重命名远程仓库的名字
git remote -v 查看所有的名字以及对应的远程仓库URL
git status 查看状态
git log 查看提交历史
git fetch [name] 从名字对应的远程仓库拉取你没有的数据
git pull 将数据合并
git push -u [name] [branch] 把本地数据推送到名字对应的远程仓库的分支
